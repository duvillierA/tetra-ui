Code

All the styles, colors, sizes are defined by your CSS rules: no need to modify the global bitmap sprite everytime!

<i class="vicon">c</i>

This is the simple tag to display an icon, where c is the character (icon) you want to display. Then use your own styles on top of this.

Preventing screen readers from reading icons:

<i class="vicon" aria-hidden="true">c</i>

Preventing Google Translate from translating icons:

<i class="vicon notranslate">c</i>
