Switch Buttons

This component implements Yes No switch buttons. The first sample is implemented with simple links and has a default value set to yes, the second with hidden radio buttons and with a default value set to no.

Yes No
Yes is activated by default 
<div class="btn-yn bootnode btn-no"
  data-view="comp_yesnobutton/yesno"
  data-event="click"
  data-url="yesno-model-fake.json.jsp"
  data-param="yesnoSample2">
  
  <a href="javascript:void(0);" class="lnk-yes">
    Yes
  </a>
  <a href="javascript:void(0);" class="lnk-no">
    No
  </a>
</div>
<div class="lbl">
  <span class="lbl-no">No by default</span>
</div>
No is activated by default 
<div class="btn-yn bootnode btn-yes"
  data-view="comp_yesnobutton/yesno"
  data-event="click"
  data-url="yesno-model-fake.json.jsp"
  data-param="yesnoSample1">
  
  <label class="lnk-yes">
    <input type="radio"
      name="btn-yn-sample" value="true" />
    It's Yes
  </label>
  <label class="lnk-no">
    <input type="radio"
      name="btn-yn-sample" value="false"
      checked="checked" />
    It's No
  </label>

</div>
<div class="lbl">
  <span class="lbl-no">No by default</span>
</div>

Ajax Tabs

Set tab content via Ajax for the tabs by setting an href value in the tab links. While the Ajax request is waiting for a response, the tab content shows a loader, then displays the tab content once loaded.

  • Sample 1
  • Sample 2
  • Sample 3

Default content

It can be a server side include for example.

loading...

<ul class="nav nav-tabs bootnode"
  data-event="click"
  data-view="comp_navtabs/navtabs"
  data-target-id="tab-sample-container"><!-- The id of the element where tab contents will be loaded -->
  
  <li class="active"><a href="/tetra-ui/doc/tab-sample-1.html">Sample 1</a></li>
  <li><a href="/tetra-ui/doc/tab-sample-2.html">Sample 2</a></li>
  <li><a href="/tetra-ui/doc/tab-sample-3.html">Sample 3</a></li>
</ul>

<div class="nav-tabs-section" id="tab-sample-container">

  <!-- Tab contents are loaded inside this tag -->
  <div class="section">
    <!-- Content displayed by default -->
  </div>
  
  <!-- If you want to display a loader, use this element -->
  <div class="loader">

    <!-- Here, put whatever you want -->
    <img src="/tetra-ui/img/loaders/ajax-loader.gif" />
    <p>loading...</p>

  </div>
</div>
