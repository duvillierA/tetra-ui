You may notice that the loading button has an extra <span> element inside it. This is in order to preserve the width of the button, so that it won't be disturbing for the user. The extra <span> element will not change the visual appearance of the other button states, so when working on dynamic buttons involving the loading state it is recommended that you use this extra element in all states.

Ajax Tabs

Set tab content via Ajax for the tabs by setting an href value in the tab links. While the Ajax request is waiting for a response, the tab content shows a loader, then displays the tab content once loaded.

  • Sample 1
  • Sample 2
  • Sample 3
<%@ include file="tab-sample-1.jsp"%>

loading...

<ul class="nav nav-tabs bootnode"
  data-event="click"
  data-view="comp_navtabs/navtabs"
  data-target-id="tab-sample-container"><!-- The id of the element where tab contents will be loaded -->
  
  <li class="active"><a href="tab-sample-1.jsp">Sample 1</a></li>
  <li><a href="tab-sample-2.jsp">Sample 2</a></li>
  <li><a href="tab-sample-3.jsp">Sample 3</a></li>
</ul>

<div class="nav-tabs-section" id="tab-sample-container">

  <!-- Tab contents are loaded inside this tag -->
  <div class="section">
    
    <!-- Content displayed by default -->
    <%@ include file="tab-sample-1.jsp"%>
    
  </div>
  
  <!-- If you want to display a loader, use this element -->
  <div class="loader">

    <!-- Here, put whatever you want -->
    <img src="/v_img/profilev2/ajax-loader.gif" />
    <p>loading...</p>

  </div>
</div>
